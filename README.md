# booking_hotel
 <img width="1900" height="868" alt="image" src="https://github.com/user-attachments/assets/23e01f2f-958f-438e-8f97-97388cfc4ada" />
Hệ thống quản lý khách sạn toàn diện được xây dựng bằng **PHP thuần** và **MySQL**, với giao diện quản trị hiện đại (Modern Dashboard UI).

## Tính năng nổi bật

### Admin Panel
* **Dashboard Analytics:** Biểu đồ doanh thu (Chart.js), thống kê booking, user.
* **Booking Management:** Xử lý đặt phòng, xếp phòng, hoàn tiền.
* **Room Management:** CRUD phòng, quản lý ảnh, tiện ích.
* **CMS:** Cài đặt thông tin website, slider, shutdown mode.
* **User & Reviews:** Quản lý người dùng và đánh giá.

### User Side
* Tìm kiếm & Đặt phòng.
* Quản lý lịch sử đặt phòng.
* Đánh giá dịch vụ.

## Công nghệ sử dụng
* **Backend:** PHP, MySQL.
* **Frontend:** HTML5, CSS3, Bootstrap 5.
* **JS Library:** Chart.js, SweetAlert (nếu có dùng alert đẹp).
* **Design:** Poppins Font, Bootstrap Icons, Dark Slate Theme.

## Cài đặt
1.  Import file `database.sql` vào MySQL.
2.  Cấu hình file `admin/component/db_config.php`.
3.  Truy cập `/admin` để vào trang quản trị.
